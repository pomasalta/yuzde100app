# %de100 - Yüzde Hesaplama Aracı

![%de100 Logo](icons/icon-192x192.png)

Profesyonel, hızlı ve kullanıcı dostu bir yüzde hesaplama uygulaması. PWA (Progressive Web App) teknolojisi ile çalışır, internet bağlantısı olmadan da kullanılabilir.

## 🚀 Özellikler

- **📊 Oran Bulma**: A sayısı, B sayısının yüzde kaçı?
- **💰 Değer Bulma**: Bir sayının belirli yüzdesi kaç?
- **📈 Değişim Hesaplama**: İki değer arasındaki yüzde değişim
- **🏷️ İndirim Hesaplama**: İndirimli fiyat ve indirim tutarı
- **➕ Yüzde Ekleme**: KDV veya komisyon ekleme
- **🔄 KDV Çıkarma**: KDV dahil tutardan KDV hariç tutar hesaplama

## 📱 Kurulum

### iPhone / iPad (Safari)
1. Safari'de siteyi açın
2. Paylaş butonuna basın (kare içinde ok)
3. "Ana Ekrana Ekle" seçeneğine dokunun
4. Uygulama ana ekranınıza eklenecek!

### Android (Chrome)
1. Chrome'da siteyi açın
2. Menü (⋮) → "Uygulama yükle" veya "Ana ekrana ekle"
3. "Yükle" butonuna basın

## 🛠️ Teknolojiler

- HTML5
- CSS3 (Dark Mode desteği)
- Vanilla JavaScript
- Service Worker (Offline çalışma)
- Web App Manifest
